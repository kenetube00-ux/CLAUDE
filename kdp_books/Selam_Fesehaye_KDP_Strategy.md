# 4. FORMATTING REQUIREMENTS

## PAPERBACK:
- Trim Size: 8.5 x 8.5 inches (square - ideal for children's picture books)
- Interior: Full color (premium color)
- Paper: White (brighter for illustrations)
- Bleed: YES - 0.125" on all sides
- Margins: Inside 0.75", Outside 0.5", Top/Bottom 0.5"
- Font: Body = Lora or Quicksand 20pt; Titles = Fredoka One or Baloo 2
- Cover Finish: Glossy

### Cover Design Brief:
FRONT: Sleepy toddler with stuffed lamb, stars + moon, heavenly glow. Navy background, golden stars, lavender/peach accents. Large rounded title font (white/gold). Badge: "30 Bible Stories & Prayers"
SPINE: Title + Author on navy background
BACK: Description, 3 circular illustration previews, "Ages 1-3" badge, barcode

## KINDLE eBOOK:
- Format: Fixed-layout (required for illustrated children's books)
- Dimensions: 1400x1400px (square) or 1400x1750px
- File Type: KPF via Kindle Create
- Resolution: 300 DPI minimum
- File Size: Under 50MB
- Cover: 2560x2560px at 300 DPI
