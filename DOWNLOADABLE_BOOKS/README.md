## KDP Upload Instructions

1. Go to [kdp.amazon.com](https://kdp.amazon.com)
2. Click "Create" > "Paperback"
3. Fill in book details (title, author, description)
4. Upload the PDF as your **manuscript interior**
5. Design or upload a cover (not included - create separately)
6. Set trim size to match (see table above)
7. Set pricing and publish!
