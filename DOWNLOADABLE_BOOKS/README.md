# KDP Books & Covers - By Daniel Tesfamariam

## HOW TO DOWNLOAD

GitHub cannot preview PDF files (you'll see "Unable to render code block" — this is normal).

### To download a file:
1. Click on any `.pdf` or `.zip` file in this folder
2. Click the **"Download raw file"** button (the download arrow icon on the right side)
3. The file will download to your computer

### Download Everything at Once:
- **All 5 Book Interiors (ZIP):** Click `ALL_5_KDP_BOOKS_Daniel_Tesfamariam.zip` then download
- **All 5 Book Covers (ZIP):** Click `ALL_COVERS_Daniel_Tesfamariam.zip` then download

### Or Download the Entire Repository:
1. Go to the main repository page
2. Click the green **"Code"** button
3. Click **"Download ZIP"**
4. Everything will be in the downloaded ZIP

## KDP Upload Instructions

1. Go to [kdp.amazon.com](https://kdp.amazon.com)
2. Click **"Create"** > **"Paperback"**
3. Fill in: Title, Author (Daniel Tesfamariam), Description
4. Upload the **Book PDF** as your manuscript (interior)
5. Upload the **Cover PDF** as your cover
6. Set trim size:
   - Books 1, 2, 4 = **8.5 x 11 inches**
   - Books 3, 5 = **6 x 9 inches**
7. Set pricing and publish!
